Welcome to MEMO GIT !

Installation de Git depuis le site officiel.

Configuration de Git

Activation des couleurs

$ git config --global color.ui true

Identification de l'utilisateur

$ git config --global user.name username $ git config --global user.email me@mail.com

NB : Ces actions entraîneront la création d'un fichier .gitconfig à la racine du dossier utilisateur.

Initialiser Git sur un projet

$ git init

NB : Cette action entraînera la création d'un dossier .git.

Faire un état des lieux du projet

$ git status

Obtenir le journal des actions du projet

$ git log

Sauvegarder le travail

Sélectionner le(s) fichier(s) à enregistrer

$ git add filename

Enregistrer les fichiers

$ git commit -m "init project"

NB : Il est possible d'utiliser le wildcard '*'.

Créer une branche

$ git branch branchname

NB : Il est conseillé de créer une branche par fonctionnalité.

Lister les branches

$ git branch

Changer de branche

$ git checkout branchname

Fusionner une branche avec la branche principale

Se positionner dans la branche qui va recevoir le travail effectué

$ git checkout oldbranchname

Fusionner les travaux des deux branches

$ git merge newbranchname

Supprimer la branche qui ne sera plus utilisée (facultatif)

$ git branch -d newbranchname

NB : Il est possible d'utiliser le wildcard '*'.

Collaborer sur un projet avec Github (dépôt distant)

Créer un repository

Choisir un nom et une desciption... puis créer !

Associer le repository au dossier local

$ git remote add origin urlduprojetgithub

Vérifier les dépôts distants du projet

$ git remote

Déposer les sources sur le repository

$ git push origin master

origin = label du dépôt

master = nom de la branche
